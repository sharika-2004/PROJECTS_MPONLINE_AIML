# Task Management System

## Description
A Java Swing based Task Management System developed using JDBC and MySQL.

## Technologies Used
- Java
- Swing
- JDBC
- MySQL
- Eclipse IDE

## Features
- User Login
- Add New Task
- Store Tasks in MySQL Database
- View Task Completion Status

## Database
Database Name:
task_management

Database Script:
task_management.sql

## Setup Instructions

1. Import the project into Eclipse.
2. Import the `task_management.sql` file into MySQL.
3. Update the database credentials in `DBConnection.java`.
4. Run `LoginUI.java`.
